E6K Pullup Resistor

 

On the Haltech E6K, there is a resistor that needs to be removed in order for the unit to work with the 3.4 DOHC DIS ignition system. Below is an image showing the resistor (R121) that needs to be removed. The best suggestion is to use a soldering iron, preferably for surface mount components, and heat slowly to avoid any collateral heating and loosening of components. This is the only modification to the board that needs to be done to make it work properly.
